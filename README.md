# Vipul Tayde - Portfolio Website

This is a personal portfolio website built with **React**, **Vite**, and **Tailwind CSS**. It showcases my experience, projects, and skills as a Computer Science Engineering student and developer.

## Features

- **Modern Tech Stack**: Built with React 19 and Vite for fast development and optimized builds.
- **Responsive Design**: Fully responsive layout using Tailwind CSS.
- **Dark Mode**: Integrated dark/light mode toggle with persistence.
- **Animations**: Smooth reveal animations using `framer-motion`.
- **Component-Based**: Modular structure for easy maintenance and expansion.

## Getting Started

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1.  Clone the repository (if applicable) or navigate to the project directory:
    ```bash
    cd portfolio-web
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

### Running the Project

To start the development server:

```bash
npm run dev
```

The site will be available at `http://localhost:5173`.

### Building for Production

To create a production build:

```bash
npm run build
```

The built files will be in the `dist/` directory, ready for deployment.

## Project Structure

- `src/components/`: Reusable UI components (Navbar, Hero, About, etc.).
- `src/App.jsx`: Main application layout.
- `public/`: Static assets like images and resumes.
